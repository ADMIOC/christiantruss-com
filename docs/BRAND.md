# Brand

ChristianTruss.com uses a dark editorial luxury system:

- Ink: `#0f0e0c`
- Cream: `#f7f4ef`
- Gold: `#b8954a`
- Teal: `#2a6b5e`
- Purple accent: `#4a3f6b`

Typography:
- Display and headings: Cormorant Garamond
- Body: DM Sans
- Labels: DM Mono

Voice:
- Authoritative but approachable
- Educational and culturally fluent
- Confidence-forward
- Never stiff or press-release-like
